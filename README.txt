WEBFORCE THEME - WordPress
============================

Um tema WordPress leve, moderno e responsivo, criado para apresentar uma agência fictícia de desenvolvimento web: a WebForce. Este tema utiliza ACF, Contact Form 7, animações com AOS e uma estrutura limpa com CSS e JS customizados.

----------------------------
REQUISITOS
----------------------------

- WordPress 6.0 ou superior
- PHP 7.4 ou superior
- Tema instalado em um ambiente WordPress (ex: LocalWP, XAMPP, etc.)

Plugins recomendados:
- Advanced Custom Fields (ACF)
- Contact Form 7
- AOS (Animate on Scroll) via CDN

----------------------------
INSTALAÇÃO DO TEMA
----------------------------

1. Faça o upload do diretório do tema `darkside-theme` para:
   `wp-content/themes/`

2. Ative o tema pelo painel do WordPress:
   - Vá para Aparência > Temas
   - Clique em "Ativar" no tema WebForce (darkside-theme)

3. Instale os plugins necessários:
   - ACF
   - Contact Form 7

4. Importe ou crie os campos personalizados (ACF):
   - Vá para: Campos personalizados > Adicionar novo
   - Configure os campos como `service_1_title`, `service_1_description`, etc.
   - Relacione ao modelo `Página` se necessário

5. Crie as páginas:
   - Home (configure como página inicial em Configurações > Leitura)
   - Sobre a WebForce
   - Fale Conosco
   - Nossos Serviços

6. Configure o menu:
   - Vá para Aparência > Menus
   - Crie um menu e atribua ao local `menu-1`

----------------------------
RECURSOS DISPONÍVEIS
----------------------------

- Página inicial com animações suaves (AOS)
- Formulário funcional com Contact Form 7
- Scroll suave com JavaScript
- Menu responsivo
- Seções com conteúdo gerenciado via ACF
- Estilização em `style.css`
- Scripts em `js/navigation.js` e `js/customizer.js`

----------------------------
AJUSTES OPCIONAIS
----------------------------

- Editar cores no `style.css`
- Adicionar responsividade extra no final do CSS com media queries
- Implementar melhorias em JavaScript para interações (ex: menu mobile)

----------------------------
ATUALIZAÇÕES RECENTES
----------------------------

- Ajustes em arquivos de scripts e estilos para melhor desempenho.
